Question: What is React? https://reactjs.org/

For developing large web applications.

  • Enforce consistent dev practice
  • Introduce some built-in convenient JavaScript logic
    • Data-binding: Normally we create a variable in code, which represents the user input (textbox), React has built-in support to always sync the user input (in textbox) with the variable’s value. Without React, plain JavaScript requires 10-20 lines to implement this sync.

<span />

Used to do inline styling. For example, in a paragraph, we want to highlight the third sentence, or we want to use bold+red font in some words in the first sentence. Then we use to control the specific word’s style.
